# Release Notes

## 小版本: 例如, 2.0.0

### 功能和增强

#### 功能名称

提供新功能或增强功能的简要介绍。

### 废弃

#### 功能名称

提供废弃功能的简要介绍，包括废弃原因和建议的替代方案。

### 将被淘汰的功能

#### 功能名称

提供将被废弃功能的简要介绍，包括废弃原因和建议的替代方案。

### 修复的问题

修复的 bug 列表可以使用文档框架提供的 `release-notes-for-bugs` 组件从 JIRA 问题中自动生成。

有关使用说明，请参阅 [自动生成 Release Notes 指南](https://product-doc-guide.alauda.cn/guide/auto_gen_releasenotes.html)。

### 已知问题

与 bug 修复类似，已知问题可以使用文档框架提供的 `release-notes-for-bugs` 组件从 JIRA 问题中自动生成。

有关使用说明，请参阅 [自动生成 Release Notes 指南](https://product-doc-guide.alauda.cn/guide/auto_gen_releasenotes.html)。
